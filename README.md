# SpineAge
The data and code for the calculation of SpineAge and related analysis.

"Result" is the original data extrated from NHANES.
'Result1' is the data after recoding for subsequent process.
'Result2' is the data after inclusion and exclusion.
'Result3' is the final version of data for subsequent feature engineering, modeling and statistical analysis.
'Code' is the collection of the code for this project, containing code for all above process.
